# 🕹️ Cyberpunk Snake  

![Game Screenshot](preview.png)  

A neon-styled Snake game with WASD controls and electronic music.  

## ▶️ How to Play  
- **WASD** to move the snake  
- Collect the glowing orbs to grow  
- Don't hit walls or yourself!  

## 🎮 Controls  
| Key | Action |  
|-----|--------|  
| W   | Up     |  
| A   | Left   |  
| S   | Down   |  
| D   | Right  |  
| Space | Pause |
